# CHANGELOG

## v2.0.2 - 16 de enero de 2026

### Nuevas características

- **Guía de inicio rápido**: Se ha añadido una nueva guía de inicio rápido (`docs/guia_inicio_rapido_jerkjs.md`) que explica cómo usar el framework JERKJS, incluyendo:
  - Formato correcto del archivo `routes.json`
  - Estructura de controladores compatible con `routes.json`
  - Uso del motor de plantillas de JERK
  - Sistema de hooks con tipos y ejemplos
  - Configuración del puerto y servidor
  - Controladores que extienden ControllerBase
  - Errores comunes y soluciones
  - Recursos útiles

- **Ejemplo avanzado de hooks**: Se ha añadido un nuevo ejemplo completo en `examples/hooks/` que demuestra:
  - Uso avanzado del sistema de hooks
  - Implementación de diferentes tipos de hooks (acciones y filtros)
  - Registro de eventos de solicitud y respuesta
  - Integración con el sistema de logging
  - Controladores con diferentes funcionalidades (usuarios, productos, autenticación)
  - Vista de ejemplo con motor de plantillas

### Mejoras

- Documentación mejorada con ejemplos prácticos
- Ejemplos más completos y explicativos
- Guía de solución de errores comunes
- Mejores prácticas documentadas

### Correcciones

- Documentación clara sobre el formato correcto de `routes.json`
- Explicación detallada de la estructura de controladores
- Solución a problemas comunes de implementación
